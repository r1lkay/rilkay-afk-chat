# rilkay-afk-chat
​
## EN
### Description
Simple lightweight plugin that sends a message to the chat when a player remains stationary for a certain amount of time, and sends a message when the player is no longer AFK. You can easily configure the delay in config.yml.​

Plugin supports MiniMessage formatting. 
Example: _<#ffffff>_

## RU
### Описание
Плагин просто отправляет сообщение в чат когда игрок стоит неподвижно определенное количество времени, а также отправляет сообщение когда игрок больше не АФК. Время через которое игрок будет считатся АФК легко меняется в config.yml. 

Плагин поддерживает способ форматирования MiniMessage.
Пример: _<#ffffff>_

## Other
Commands:
/rafk reload - reloads the plugin
Permissions:
rilkayafk.reload - allows to use the reload command
